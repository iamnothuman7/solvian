Sobre o RADIAX

Sistema All-in-One para empresas de energia solar. Desenvolvido como TCC em Sistemas de Informação.

💡 Visão

O que é o RADIAX?

O RADIAX é o MVP de uma startup B2B que oferece uma plataforma completa para empresas de energia solar — do dimensionamento técnico à manutenção preditiva com Inteligência Artificial.

🎯

Missão

Democratizar o acesso a ferramentas profissionais de dimensionamento solar, usando dados reais de satélite e inteligência artificial.

🔬

Diferencial

Integração direta com os dados da NASA POWER para obter irradiação solar precisa para qualquer localização geográfica do planeta.

🚀

Futuro

Expandir para um SaaS completo com CRM integrado, controle de estoque, monitoramento IoT e marketplace de equipamentos.
